Para correr el programa en cmd tuvimos que hacer un jar
Se corre con 
java -jar DatosProyecto2Final.jar
Esperar un rato para la primera configuracion porque tarda por crear la base de datos.

Se puede probar con usuarios 
usuario: andres
contrasena: andres123
usuario: robert
contrasena: robert123
usuario: felipe
contrasena: felipe123
O ya bien se puede crear su propio usuario.

La recomendacion puede ser dada por 
-búsqueda de películas y similitudes. Boton 2
-búsqueda de personas top con quienes se comparten peliculas. Boton 3
-agregar peliculas a su propia lista para sugerir en base a sus propias preferencias. Boton 4
